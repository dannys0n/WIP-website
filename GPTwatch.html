GPT watch app Prototype

ChatGPT-like experience for smart watches,
With keyboard and voice input.

An unreleased personal project created for smart watches running WearOS. Utilizing OpenAI's API and a network connection, the assistant generates text running on the latest models

tools & resources used

  • Android Studio.
  • Kotlin.
  • OpenAI API.
  • Retrofit.
  • WearOS and Documentation.
  • GitHub.

Persistent Memory

During A chat session, the assistant will remember previous messages between user and assistant to be context aware using a basic list.

Code snippit of previous conversations being used with new user prompt to generate response

Formatting

messages get formatted for Markdown and basic LaTeX (bullets, numbered lists, headers, inline equations, etc).

Code snippit of Markdown using Markwon and locally preproccessing LaTeX formatting for UI

Lightweight

The apk is kept at the lowest file size as reasonablly possible by integrating Proguard to shrink, optimize, and obfuscate.
